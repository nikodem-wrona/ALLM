# ALLM

All llms client

## Configuration

Api keys for providers are stored in app configuration locally, with following structure:

```json
{
  "providers": {
    "openai": {
      "apiKey": ""
    }
    // other providers
  }
}
```

configuration can be found users home folders:

### MacOS

```shell
/Users/$user/.config/allm/config.json
```
